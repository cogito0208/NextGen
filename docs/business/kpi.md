# 기대 효과 및 KPI

## 1. 직접적 재무 수익 극대화 (Profitability)

### 1.1 장비 가동률(Utilization) 혁신

#### 현황
- 국내 도비 업체 평균 가동률: **65~70%**
- 개선 목표: **85%** (15% 향상)

#### 재무 효과
**예시**: 대당 월 매출 2,000만 원 × 장비 10대 보유 기업
```
현재 매출: 2,000만 원 × 10대 × 70% = 월 1억 4,000만 원
목표 매출: 2,000만 원 × 10대 × 85% = 월 1억 7,000만 원
월간 증가: 3,000만 원
연간 증가: 3.6억 원
```

#### 구현 방법
- AI 배차 최적화 알고리즘
- 실시간 장비 가용성 트래킹
- 유휴 시간 자동 알림
- 최적 경로 제안

### 1.2 견적 누수 차단

#### 현황
- 수기 견적 시 오차 발생률: **8~12%**
- 주요 누수 항목:
  - 운송비 누락
  - 특수 장비 할인 미적용
  - 안전장치 비용 미반영
  - 추가 인력 산정 오류

#### 재무 효과
- **마진율 3~5% 안정적 향상**
- 연매출 20억 기업: 연간 6,000만~1억 원 추가 이익

#### 구현 방법
- AI 자동 견적 생성 (10초 이내)
- 과거 데이터 기반 가격 최적화
- 체크리스트 자동 검증
- 다중 가격 티어 자동 적용

### 1.3 미수금 회수 속도 향상

#### 현황
- 평균 미수금 회수 기간(DSO): **60일**
- 목표 DSO: **48일** (20% 단축)

#### 재무 효과
- 현금 흐름 개선
- 운전자본 효율 증대
- 금융 비용 절감

#### 구현 방법
- 프로젝트 완료 즉시 자동 인보이싱
- 미수금 알림 자동화 (7일, 14일, 30일)
- 고객 포털 실시간 확인
- 디지털 결제 연동

## 2. 운영 비용 및 리소스 절감 (Efficiency)

### 2.1 배차/행정 시간 70% 단축

#### AS-IS 워크플로우
```
06:00 - 관리자 출근
06:30 - 엑셀에서 당일 작업 확인
07:00 - 기사 30명에게 전화/문자 (30분 소요)
07:30 - 장비 배정 엑셀 업데이트
08:00 - 고객사에 작업 시작 알림
```
**총 소요 시간**: 2시간/일 = **40시간/월**

#### TO-BE 워크플로우
```
06:00 - 관리자 출근
06:05 - 플랫폼에서 AI 배차 확인
06:10 - 드래그앤드롭으로 미세 조정
06:15 - 일괄 알림톡 발송 (자동)
06:20 - 완료
```
**총 소요 시간**: 20분/일 = **7시간/월**

#### 효과
- **시간 절감**: 33시간/월 (82.5%)
- **인건비 환산**: 월 165만 원 (시급 5만 원 기준)
- **처리량 증가**: 1명이 3배 업무 처리 가능

### 2.2 서류 작업 제로화

#### AS-IS
- 작업계획서 작성: 30분/건
- 장비제원표 정리: 20분/건
- 안전점검표 작성: 15분/건
- 일일 보고서 작성: 40분/일

**월 30건 프로젝트**: 약 **60시간**

#### TO-BE
- 작업계획서: 1클릭 생성 (1분)
- 장비제원표: 자동 연동
- 안전점검표: AI 자동 작성
- 일일 보고서: 실시간 자동 생성

**월 30건 프로젝트**: 약 **3시간** (95% 절감)

### 2.3 총 운영비용 절감 효과

| 항목 | AS-IS | TO-BE | 절감액 (월) |
|------|-------|-------|-------------|
| 배차 인건비 | 400만 원 | 120만 원 | 280만 원 |
| 사무 인건비 | 350만 원 | 100만 원 | 250만 원 |
| 통신비 (전화/문자) | 80만 원 | 20만 원 | 60만 원 |
| 서류/인쇄비 | 40만 원 | 5만 원 | 35만 원 |
| **합계** | **870만 원** | **245만 원** | **625만 원** |

**연간 절감**: **7,500만 원**

## 3. 중대재해처벌법 대응 및 리스크 방지 (Compliance)

### 3.1 사고 예방을 통한 보험료 인하

#### 현황
- 도비 업체 평균 사고율: **연 3~5건** (업계 평균)
- 기업 보험료: 연 **2,000~3,000만 원**

#### 목표
- 사고율 40% 감소: 연 **1.8~3건**
- 3년 무사고 시 보험료 인하율: **15~25%**

#### 재무 효과
- 보험료 절감: 연 **300~750만 원**
- 3년 누적: **900만~2,250만 원**

#### 구현 방법
- 3D 리프트 시뮬레이션 (사전 위험 감지)
- AI 실시간 안전 모니터링
- TBM (Tool Box Meeting) 자동 기록
- 작업자 피로도 관리

### 3.2 디지털 안전 증빙 확보

#### 법적 가치
중대재해처벌법 제4조 (사업주의 의무):
> "사업주는 안전보건관리체계를 구축하고, 그 이행에 관한 조치를 하여야 한다"

#### 증빙 자료
1. **AI 안전 점검 로그**
   - 모든 작업 전 안전모 착용 확인
   - 장비 결속 상태 자동 검증
   - 타임스탬프 기록

2. **3D 시뮬레이션 데이터**
   - 작업 전 리스크 분석 기록
   - 대안 시나리오 검토 증빙
   - 최적 방안 선택 근거

3. **교육 이력**
   - 작업자별 안전 교육 수료 기록
   - 정기 교육 자동 알림 이력
   - 자격증 만료 관리 로그

#### 법적 효과
- 사고 발생 시 **면책 자료** 확보
- 검찰/노동부 조사 시 **입증 책임** 충족
- 형사처벌 위험 최소화

### 3.3 컴플라이언스 비용 절감

#### AS-IS
- 안전 컨설턴트 비용: 월 200만 원
- 수기 문서 작성: 월 80시간
- 감사 대응 준비: 분기당 1주일

#### TO-BE
- 자동 리포트 생성
- 실시간 컴플라이언스 체크
- 감사 자료 1클릭 추출

**연간 절감**: 약 **3,000만 원**

## 5. 비재무적 효과

### 5.1 고객 만족도 향상
- 실시간 작업 현황 공유
- 투명한 비용 내역
- 빠른 문제 해결
- **NPS 점수**: 40 → 65 목표

### 5.2 직원 만족도 향상
- 불필요한 야근 감소
- 명확한 업무 프로세스
- 성과 가시화
- **이직률**: 30% → 15% 목표

### 5.3 브랜드 가치 상승
- "디지털 혁신 선도 기업" 이미지
- 갑사(대기업)와의 파트너십 강화
- 업계 Best Practice 사례
- 신규 고객 유입 증가

### 5.4 확장성 확보
- 일본 시장 진출 기반 마련
- 데이터 기반 의사결정 문화
- M&A 시 기업가치 상승

## 6. KPI 추적 체계

### Daily KPIs
- 장비 가동률 (%)
- 당일 프로젝트 진행률
- 안전 위험 알림 건수

### Weekly KPIs
- 견적 전환율
- 평균 배차 소요 시간
- 직원 생산성 지수

### Monthly KPIs
- MRR (Monthly Recurring Revenue)
- DSO (Days Sales Outstanding)
- NPS (Net Promoter Score)
- 사고 발생 건수

### Quarterly KPIs
- 고객 이탈률 (Churn Rate)
- 신규 고객 획득 비용 (CAC)
- 고객 생애 가치 (LTV)
- 장비 ROI
