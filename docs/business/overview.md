# KMTLS Next-Gen 프로젝트 개요

## 1. 프로젝트 정의
**AI 기반 중량물·도비 통합 운영 플랫폼**

고중량물 취급 및 크레인 운영의 전 과정을 디지털화하여 안전성, 효율성, 수익성을 극대화하는 엔터프라이즈 SaaS 플랫폼

## 2. 프로젝트 배경

### 2.1 외부 환경: 규제 강화와 산업 구조의 변화

#### 중대재해처벌법(SAPA) 대응의 필연성
- 고중량물 취급은 사고 발생 시 치명적인 인명 피해로 직결
- 기존 수기식 안전 점검: 법적 소명 자료로서의 신뢰도 낮음
- 실시간 위험 감지에 한계
- **필요**: 디지털 안전 증빙 체계

#### 숙련공 부족 및 기술 단절
- KMTLS 내 정규직 베테랑 기술자의 노하우가 디지털화되지 않음
- 비정규직 및 신입 인력 투입 시 작업 퀄리티 편차 발생
- 안전 리스크 증대
- **필요**: AI 기반 표준화된 작업 절차(SOP)

### 2.2 내부 현황: '아날로그 워크플로우'의 한계

#### 데이터 파편화
- 현장 인력과 대형 장비 스케줄: 엑셀과 PPT에 머물러 있음
- 실시간 가동률 파악 불가능
- 최적 배치 불가능
- **필요**: 통합 데이터 플랫폼

#### 커뮤니케이션 비용
- 갑사(대기업 고객)의 보안 구역 내 작업 시 정보 전달 지연
- 무전 의존도 높음
- 작업 효율 저하
- 정산 과정의 불필요한 마찰
- **필요**: 실시간 디지털 커뮤니케이션 체계

## 3. 전략적 가치

### 3.1 법적 리스크 제로화

#### 디지털 안전 증거력 확보
- AI가 기록한 TBM 이력
- On-device 안전 모니터링 데이터 (폐쇄형 AI 솔루션)
- 사고 예방 + 유사시 기업 보호
- 보안 가이드 준수하면서도 업계 최고의 기술력 증명

#### 표준화된 작업 절차(SOP) 강제
- AI 가이드를 통해 현장 인력의 숙련도와 관계없이 동일한 수준 유지
- 안전 및 작업 품질 표준화

### 3.2 운영 효율성 및 수익성 극대화

#### 장비 가동률 목표 달성
- 지능형 배정 알고리즘
- 유휴 장비 최소화
- 공장 단지 내 이동 동선 최적화
- 시간당 생산성 향상

#### 사무 공수 70% 절감
- 엑셀 배정 자동화
- 영수증 처리 자동화
- PPT 보고서 작성 자동화
- 고부가가치 기획 업무에 집중

## 4. 핵심 지표 (KPI)

| 항목 | AS-IS | TO-BE | 가치 |
|------|-------|-------|------|
| 안전 관리 | 사후보고, 수기기록 | 실시간 탐지, 디지털 로그 | 디지털 안전 증빙 확보 |
| 인력 운용 | 경험기반 수동배정 | 데이터기반 최적배정 | 숙련도 편차 극복(SOP) |
| 장비 관리 | 유선확인, 가동정지, 기산 발생 | IoT 실시간 트레킹 | 가동률 및 자산수익률 증대(ROA) |
| 고객 신뢰 | 사후정산, 분쟁 | 실시간 대시보드 공유 | 갑사와의 전략적 파트너십 강화 |

## 5. 타겟 사용자

### Primary Users
- **현장 관리자**: 프로젝트 실행, 안전 점검
- **배차 담당자**: 장비/인력 스케줄링
- **영업팀**: 견적, CRM, 고객 관리

### Secondary Users
- **경영진**: 대시보드, KPI 모니터링
- **재무팀**: 회계, 인보이싱
- **안전 담당자**: 컴플라이언스, 보고서

### External Users
- **갑사(고객)**: 고객 포털 웹사이트를 통한 프로젝트 모니터링
- **현장 작업자**: 모바일 앱을 통한 작업 확인 및 보고

## 6. 기술 스택

- **Frontend**: Next.js 16.2, React 19
- **Styling**: Stitches CSS-in-JS

- **Design System**: Google Stitch MCP 통합
