# 기능 명세서 (Features)

## 개요
KMTLS Next-Gen의 14개 주요 모듈 상세 명세

## 파일 구조

### 작성 완료 (상세)
- `dashboard.md` - 대시보드 & 분석
- `project.md` - 프로젝트 관리 (작성 예정)
- `fleet.md` - 장비·자산 관리 (작성 예정)

### 작성 필요 (템플릿 기반)
나머지 모듈은 `_template.md`를 복사하여 작성:

```bash
cp _template.md crm.md
cp _template.md quote.md
# ... (나머지)
```

- 예: 대시보드 작업 → `dashboard.md`만 (2-3K 토큰)
